# 布局更新说明

## 新布局结构
```
顶部栏 (28px)
├── 主内容区 (flex: 1)
│   ├── 左侧图层 (250px) - 加宽了
│   └── 画布区域 (flex: 1)
├── 控制区域 (auto)
│   ├── 大图标工具栏 (48px) - 图标加大
│   └── 属性面板 (auto) - 从右侧移到下方
└── 时间轴 (160px)
```

## 需要添加的CSS
- .layers-sidebar: 左侧图层侧边栏 (250px宽)
- .sidebar-layer-item: 侧边栏图层项 (更大)
- .controls-area: 控制区域容器
- .toolbar-large: 大图标工具栏
- .tb-btn.large: 大尺寸按钮
- .props更新: 横向布局，放在工具栏下方

## 改进点
1. 图层列表更宽 (120px → 250px)
2. 工具图标更大 (更好点击)
3. 属性面板移到更合理的位置
